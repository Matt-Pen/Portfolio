QVault

This is a team project, it was created as a way for students to access the previous years papers with ease instead of solely depending on going to the library or getting hold of a seniors google drive with question papers.It was made using HTML, CSS and Angular for the frontend, and the backend made using Java and MongoDB while using Vertx for the API.

  • Being part of the backend, I integrated functions for the librarians to be able to upload PDFs of question papers, to return the requested set of question paper that match the search criteria.
  • Integrated a forgot password feature where it would send an Email to the user with a password reset token using SMTP.
  • Wrote functions for bookmarking question papers so that students can access question papers they use frequently quicker.
  • Used postman for API testing MongoDB Atlas for Database.

Easy-Medicine

This is a project called Easy-Medicine, created to streamline pharmacy management by providing an easy-to-use billing system and inventory control. It was made using VB.NET for the frontend and backend and MySQL as the database.

  • Implemented the Login Module to control user access, allowing pharmacists to manage both billing and inventory while cashiers could only access billing.
  • Developed the Product Module to add, update, and delete medicines and supplier details, and view current inventory records.
  • Built the Billing Module to select medicines from a dropdown, auto-fill prices, add to cart, process cash or card payments, and deduct stock automatically.
  • Created the Reports Module to generate PDF reports for customer purchases, supplier purchases, payment history, and inventory with search filters.
